Install EPS WordPress Plugin

To add the plugin to your WordPress website from the dashboard, follow these steps:
1. Download the plugin .zip file from the repository.
2. Log in to your WordPress dashboard and go to Plugins â†’ Add New.
3. Click Choose File, select the plugin .zip file, and then click Install Now.
4. After the installation is complete, click Activate Plugin.
5. The plugin will now appear as a menu item in the WordPress dashboard.
6. Open EPS Settings.
7. Enter the required credentials (Merchant ID, Password, etc.).
8. Enable the EPS Payment System in WooCommerce.
