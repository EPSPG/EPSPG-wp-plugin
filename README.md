Install EPS WordPress Plugins

To add a plugin to your WordPress website from the dashboard:
1. In your WordPress dashboard, choose Plugins > Add new.
2. Click Choose File, locate the plugin .zip file, then click Install Now.
3. After the installation is complete, click Activate Plugin.
4. Now, Show the plugin into WordpPress Dashboard menu item.
5. EPS Setting
6. Input Credential
7. Now Enabled EPS Payment System into WooCommerce
